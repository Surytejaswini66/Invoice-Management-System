# Invoice Management System

## Project Overview
This project is a web-based application designed to manage invoices efficiently. It includes the following features:
- **User Authentication**: Secure login and sign-up functionalities.
- **Invoice Management**: Add, edit, delete, and view invoices seamlessly.
- **Data Persistence**: Uses `localStorage` to save data across sessions.
- **Responsive Design**: Optimized for desktop and mobile devices.

---

## Setup Instructions

### Prerequisites
- Node.js (v14 or later)
- A code editor like Visual Studio Code

### Steps to Setup Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Surytejaswini66/Invoice-Management-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd invoice-management-system
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
5. Open your browser and visit:
   ```
   http://localhost:3000
   ```

---

## Usage Instructions

### Accessing the Application
1. **Login/Sign-Up**:
   - New users can register using their email and password.
   - Existing users can log in to access their invoices.

2. **Invoice Management**:
   - **Create Invoice**: Use the "Create Invoice" form to add new invoices.
   - **Edit Invoice**: Click on an invoice in the list to modify its details.
   - **Delete Invoice**: Remove unwanted invoices directly from the list.

3. **Navigation**:
   - Navigate between pages such as Home, Login, and Create Invoice using the navigation bar.
